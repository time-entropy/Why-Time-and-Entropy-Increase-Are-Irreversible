# Why Time and Entropy Increase Are Irreversible

In a groundbreaking framework developed by independent researcher Zou Zhi Kai, time and entropy increase are shown to be fundamentally irreversible through a discrete model of spacetime. The theory posits a topologically invariant quantum network—a fixed graph of Space—where each network state update corresponds to one discrete moment in time. Entropy is defined non-statistically as the multiplicative product of all space elementary quanta  energy values at each update. Following the thermodynamic rule of energy homogenization (energy flows from higher to lower quanta), each update mathematically guarantees an increase in entropy, establishing a direct one-to-one correspondence: one update → one entropy value → one moment in time. This constructs an irreversible temporal order from first principles.
